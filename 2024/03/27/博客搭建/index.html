    • 前提

    PC装有node.js

    Npm使用国内镜像淘宝

    1. 设置淘宝镜像源
    npm config set registry https://registry.npm.taobao.org/

    1. 设置官方镜像源

      npm config set registry https://registry.npmjs.org

    2. 查看镜像使用状态

      npm config get registry

    如果返回https://registry.npm.taobao.org/，说明配置的是淘宝镜像。

    如果返回https://registry.npmjs.org/，说明配置的是淘宝镜像。

    解决npm证书失效，nmp安装报错

    1. 清除npm缓存

      npm cache clean --force

    2. 取消ssl认证、

      npm config set strict-ssl false

    3. 重新安装

    安装Git，傻瓜式安装

    网址

    https://registry.npmmirror.com/binary.html?path=git-for-windows/

    建站

    1、初始化，创建一个项目

    hexo init <项目名称>

    2、运行项目，在该项目目录下

    Hexo server

    3、新建一片文章

    hexo new "My New Post”

    4、重新绑定端口

    Hexo server  --ip ip地址 --port 端口

    5、清除hexo缓存

    Hexo clean

    6、生成静态文件（上传github用）

    Hexo g

    7、上传到github

    Hexo d

    将项目上传到github的前置

    1、安装hexo-deployer-git插件

    Npm  install hexo-deployer-git

    运行成功会提示：INFO Delploy done:git

    2、找到deploy_git下的config文件（在.git目录下，被隐藏了）并打开，加入以下内容

    [user]
email = cj15020907347@163.com
name = XFDOG

    3、找到_config.yml文件,添加如下内容

    deploy:
  type: 'git'
  repo: https://github.com/XFDOG/XFDOG.github.io.git
  branch: master

repo：写的是github的仓库链接

    我们在本地写好一片笔记上传到github，执行三个步骤

    1、

    hexo clean

    2、

    hexo g

    3、

    hexo d

    上传成功后，github生成网站会很慢，内心等待
