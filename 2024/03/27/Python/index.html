    break和continue

    break和continue是专门在循环中使用的函数

    break：某一条件满足时，退出循环，不再执行后续重复的代码

    continue：某一条件满足时，推出本次循环，不执行后续重复的代码

    break和continue只针对当前所在的循环

    break

    i = 0
while i < 10:
    # break 某一条件满足时，退出循环，不再执行后面的代码
    if i == 3:  # 当 i == 3 时，结束循环
        break
    print(i)
    i += 1

    continue

    i = 0
while i < 10:
    # continue 某一条件满足时，退出本次循环，不再执行后面的代码
    if i == 3:  # 当 i == 3 时，不打印
        i += 1  # 不加计数器，会陷入死循环
        continue
    print(i)
    i += 1

    文件指针

    文件指针标记了从哪个位置开始读取数据

    第一次打开文件时，通常文件指针会指向文件开始的地方

    当执行了read方法之后，文件指针会移动到读取内容的末尾

    打开文件的方式

    f = open("文件名", "访问方式")

    r：以只读方式打开文件。文件的指针将会放在文件的开头，这是默认模式。如果文件不存在，抛出异常。

    w：以只写方式打开文件。如果文件存在会被覆盖。如果文件不存在，创建新文件

    a：以追加方式打开文件。如果文件以存在，文件指针将会放在文件的结尾。如果文件不存在，以创建新文件进行写入。

    r+：以读写方式打开文件。文件的指针将会放在文件的开头。如果文件不存在，抛出异常。

    w+：以读写方式打开文件。如果文件存在会被覆盖。如果文件不存在，创建新文件

    a+：以读写方式打开文件。如果文件已存在，文件指针将会放在文件的结尾。如果文件不存在，创建新文件进行写入。

    # 1、打开文件
file = open("1.txt","a")
# 2、写入文件

# 3、关闭文件

    import pygame


class GameSprite(pygame.sprite.Sprite):
    """飞机大战精灵"""

    def __init__(self, image_name, speed=1):
        # 调用父类的初始化方法
        super().__init__()
        # 定义对象的属性
        self.image = pygame.image.load(image_name)
        self.rect = self.image.get_rect()
        self.speed = speed

    def update(self):
        # 在屏幕的垂直方向上移动
        self.rect.y += self.speed
