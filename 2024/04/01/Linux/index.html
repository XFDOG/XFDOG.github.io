linux命令的基础格式

command [-options] [parameter]

command:命令本身
options:[可选，非必填]的一些选项，通过选项控制命令的行为细节 
parameter:[可选,非必填]命令的参数，多数用于命令的指向目标等
[]语法中表示可选的意思

ls命令

ls [-a -l -h] [Linux路径]
ls：直接使用ls，以平埔形式列出目录下内容
-a：显示所有文件，包括 隐藏文件
-l：以树状显示文件，包括文件的权限，所属用户和用户组，大小等
-h：通常配合-l使用，以常规方式显示文件大小

home目录

当前登录用户的home目录作为当前工作目录
每个linux操作用户在linux系统的个人账户目录，路径在/home/用户名、

直接使用cd命令，回到当前登录用户的家目录下

mkdir

创建目录
选项：
-p 可以创建多层目录

touch

创建文件，没有选项

cat

查看文件，全部显示出来

more

查看文件，可以翻页
空格下一页，B向上翻页， q退出

cp

复制文件，文件夹
选项：
-r 用于复制文件夹

参数：
被复制的文件  要复制去的地方

mv

移动文件/文件夹 ，也可以用来修改文件名

参数：
被移动的文件/文件夹  要移动去的地方

rm命令

rm  删除文件/文件夹

选项:
-r  删除目录
-f  直接删除，不需要提醒


参数：
文件或文件夹路径，可以使用通配符来删除更多文件

which

使用时linux'命令本质上都是执行二进制程序
which用来查找命令的位置

语法：
which 命令

find

查找文件或者文件名,查找文件也可以使用通配符
按照文件名搜索：
find 起始路径 -name "文件名"


按照文件大小搜索：
find  起始路径 -size +|- -n[kMG]

+代表大于，-代表小于
n代表大小数字
KMG表示大小单位，k表示kB，M代表MB，G代表GB

grep

从文件中通过关键字过滤文件行

语法：
grep [-n] 关键字 文件路径

-n：表示再结果中显示匹配的行的行号
关键字：建议用""包围起来

wc

统计文件的行数，单词，字节数量等

语法：
wc [-c -m -l -w] 文件路径

-c：统计字节数量
-m：统计字符数量
-l：统计行数
-w：统计单词数量

|管道符

将左边的结果作为右边的输入

echo

在命令行输出指定内容，最好用双引号包围
-n:输出不换行

反引号`

echo后面输出的是命令的话，可以用反引号包围，即可输出执行结果


echo `ls /`

重定向符号

> 将左边命令的结果覆盖到右侧文件
>> 将左边命令的结果追加到右侧文件

tail

查看文件尾部内容，跟踪文件的最新改动

语法：
tail [-f -num] 文件路径

-f：表示持续跟踪
-num：表示查看尾部多少行，不写默认10行

创建、删除用户

chmod修改权限

chmod 注意：只有文件、文件夹的所属用户或root用户才可以修改

语法：
chmod [-R] 权限 文件或文件夹
-R：对文件夹内的全部内容生效

例：
chmod u=rwx g=rw o=r hello.txt //将文件权限修改为rwxrw-r--

权限的数字序号

r：数字表示为4
w：数字表示为2
x：数字表示为1
0表示没有任何权限

所以rwx满权限为7

chown修改文件、文件夹的所属用户或用户组

此命令只适合root用户

语法：
chown [-R] [用户] [:] [用户组] 文件或文件夹

-R：同chmod
用户：修改所属用户
: ：用于分隔用户和用户组
用户组：修改所属用户组

主机状态

查看系统资源占用top

top的交互式命令，区分大小写

h键：显示帮助画面
c键：显示产生进程的完整命令，等同于-c
f键：选择需要展示的项目，空格选中
M键：根据驻留内存大小RES排序
P键：根据cpu使用百分比大小进行排序
T键：根据时间/累计时间进行排序
E键：切换顶部内存显示单位
e键：切换进程内存显示单位
l键：切换显示平均负载和启动时间信息
i键：不显示闲置或无用的进程，等同于-i
t键：切换cpu状态信息
m键：切换显示内存的信息

使用直接输入top命令	，默认5秒刷新一次，按q或ctrl+c退出

选项：
-p：只显示某个进程的信息
-d：设置刷新时间，默认是5s
-c：显示产生进程的完整命令，默认是进程名
-n：指定刷新次数，比如top -n 3，刷新输出3次后推出
-b：以批次的方式执行top，一般配合-n指定输出几次统计信息，将输出重定向指定文件，比如 top -b -n 3 > /tmp/top.tmp
-i:不显任何闲置或无用的进程
-u：查特定用户启动的进程

PID：进程id

%CPU：进程占用CPU率
%MEM：进程占用内存率
TIME+：进程使用CPU时间总计，单位10毫秒
COMMAND：进程的命令或名称或程序文件路径

压缩和解压缩

tar

  • .tar称之为tarball，归档文件，即简单的将文件组装到一个.tar的文件内，并没有太多文件体积的减少，仅是简单的封装
  • .gz，也常见为.tar .gz .gzip格式压缩文件，即使用压缩算法将文件压缩到一个文件内，可以极大地减少压缩后的体积

针对这两种格式，使用tar命令均可以进行压缩和解压缩的操作

语法：
tar [-c -v -x -f -z -C] 参数1 参数2 ... 参数n

-c：创建压缩文件，用于压缩模式
-v：显示压缩、解压缩的过程，用于查看进度
-x：解压模式
-f：要创建的文件，或者要解压的文件，-发、选项必须在左右选项中处于最后一个
-z：gzip模式，不使用-z就是普通的tarball模式
-C：选择解压的目的地，用于解压模式

压缩
tar -cvf test.tar  1.txt 2.txt ...
tar -zcvf test.tar.gz  1.txt 2.txt ...


解压
tar -vxf test.tar 

tar -vxf test.tar -C /home/ubuntu

zip

压缩语法：
zip -r 参数1 参数2 ...

-r：被压缩的文件中有文件夹使用-r没有不需要


解压语法：
unzip -d 参数1...
-d:指定解压目录
unzip text.zip -d /home/ubuntu
