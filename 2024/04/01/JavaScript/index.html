书写位置

外部式和内部式代码最好都写在</body>标签上方

外部式

引用外部的*.js文件。此时<script>标签内写的代码不会执行

<script src = "文件路径"></script>

内部式

<body>
*****
    <script>
        /*
        代码
        */
    </script>
</body>

内联式

<button onclick = "alert('逗你玩')">有本事点我</button>

    let name = prompt("请输入您的姓名")
</script>

输出语句

document.write

输出在页面上

<script>document.write("hellow world")</script>

alert

弹窗输出

<script>alert('hello world')</script>

console.log

控制台输出

<script>console.log('hellow world')</script>

变量的声明

js的变量命名结构与其他语言相同；但是要先声明，可以用let var const声明

let

<script> 
let num = 1
</script>

var

<script>
var num = 1
</script>

let 与 var的区别

<script>
let num = 1
let num = 2
//let的这种改变变量的方式式错误的
</script>

<script>
var num = 1
var num = 2
//var可以这样定义，而且使用var定义的变量可以先使用在定义，如下
str = "你好"
document.write(str)
var str
</script>

const

使用const定义的变量是常量，必须赋初始值，不嫩被改变

<script>
const str = "唯一"
</script>

数组的基本使用

数组的定义

获取与修改数组元素的方法与其他语言相同

<script>
let arr = []
console.log(arr)
</script>

arr.length返回数组的长度

<script>
let arr = [1，2，3，4]
console.log(arr.length)
</script>

数据类型

转义符使用规则同其他语言

字符串拼接使用+号

基本数据类型

  1. 数字型：整数或小数都属于数字型,包括正负数
  2. 字符串型:使用单引号,双引号,反引号包裹的数据都是字符串型
  3. 布尔型:true,false
  4. null、undefined

typeof(变量名) //输出变量类型

<script>
let arr = [1，2，3，4]
console.log(typeof(arr))
</script>

数字型

NaN型

<script>
let arr = 11
console.log(arr - "你好") //这种情况会出现NAN型 ,任何NAN参数的运算都会返回NaN,
 
console.log(NaN - 2) //返回NaN
  
</script>

字符串型

使用+号拼接字符串

<script>
let age  = 18

document.write('我今年'+age+'岁了')
</script>

模板字符串

使用反引号包围整个字符串

<script>
let age  = 18

document.write(`我今年${age}岁了`)
</script>
