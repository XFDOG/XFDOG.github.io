数组的基本使用

数组里面可以存放任意数据的类型

1、字面量声明数组

​ let arr = [1,2,true,’xiaoming’]

2、使用new Array()构造函数声明

​ let arr = new Array(1,2,3,4,’xiaoming’)

求数组中的最大、最小值

<script>
let arr = [3,5,6,21,76,5]
let max = arr[0]
let min = arr[0]

for(let i=1;i<arr.length;i++){
    max = max>arr[i]?max:arr[i]
    min = min<arr[i]?min:arr[i]
}
console.log(`最大值：${max},最小值:${min}`);
</script>
